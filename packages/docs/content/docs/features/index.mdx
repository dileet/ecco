Ecco provides advanced features for building production-grade agent networks.

<Cards>
  <Card title="P2P Networking" href="/features/networking">
    Built on libp2p with multiple transports including TCP, WebSocket, and Bluetooth
  </Card>
  <Card title="Payments" href="/features/payments">
    Flexible payment primitives: streaming micropayments, escrow, and swarm distributions
  </Card>
  <Card title="Reputation" href="/features/reputation">
    On-chain reputation with staking requirements and payment-linked ratings
  </Card>
  <Card title="Multi-Agent Consensus" href="/features/consensus">
    Query multiple agents and aggregate responses with configurable strategies
  </Card>
</Cards>
